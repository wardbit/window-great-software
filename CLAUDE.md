## 项目概述

这是一个 Windows 软件推荐清单项目，纯 Markdown 文档仓库，无代码编译或构建流程。

## 文件结构

- `推荐软件.md` - 核心内容文件，包含所有软件推荐
- `README.md` - 项目简介

## 内容分类

`推荐软件.md` 按以下类别组织：
1. 软件推荐（文件搜索、播放器、压缩、截图、笔记、效率工具、加密、下载等）
2. 电子书工具
3. 删除重复文件工具
4. Chrome 插件与油猴脚本
5. 翻译软件

## 编辑规范
